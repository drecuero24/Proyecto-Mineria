1.1 Particiones

Dividimos nuestro conjunto de datos en 3 subconjuntos. Los datos de entrenamiento(el 10% de los datos totales) que serán los que utilicemos para tratar y aplicar las técnicas de ciencia de datos para recoger información. Los datos de prueba(otro 10% de los datos totales) serán los datos donde probaremos que nuestras técnicas funcionan y nuestro modelo es válido, cambiando así lo realizado hasta entonces si no funciona como se espera. Por último, el mayoritario conjunto de datos de validación mostrarán finalmente lo realizado a lo largo del trabajo en el final del mismo.
